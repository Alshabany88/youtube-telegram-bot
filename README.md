# YouTube Telegram Bot 🎬

بوت تلجرام لاستخراج روابط وعناوين الفيديوهات من يوتيوب.

## المميزات ✨
- استخراج جميع روابط قوائم التشغيل
- جلب جميع فيديوهات أي قناة (حتى 20,000 فيديو)
- عرض عناوين الفيديوهات مع الروابط
- إرسال ملف نصي منظم

## الأوامر 📋
- `/start` - بدء الاستخدام
- `/help` - المساعدة
- `/about` - معلومات عن البوت
- `/channel @username` - استخراج فيديوهات قناة

## الاستخدام 🚀
- لقائمة تشغيل: أرسل الرابط مباشرة
- لقناة: أرسل @اسم_القناة
